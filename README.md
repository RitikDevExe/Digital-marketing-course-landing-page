\# Digital Marketing Landing Page



A modern \*\*Digital Marketing Course Landing Page\*\* built using \*\*pure HTML5 and CSS3\*\* by the reference of a FIGMA design.



\---



\## 🚀 Project Overview



This project focuses on creating a clean and professional landing page UI while improving frontend fundamentals like:



\- Semantic HTML structure

\- CSS Flexbox 

\- Section-based webpage design

\- Clean spacing \& typography

\- Figma to code conversion



The website includes multiple modern sections such as:



\- Hero Section

\- Course Highlights

\- Career Opportunities

\- Certification Section

\- Curriculum Section

\- Testimonials

\- FAQs Accordion UI

\- Company Logos

\- Footer



\---
